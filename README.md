# PhonePe-Transaction-Analysis-Dashboard
Power BI project, where I analyzed PhonePe transaction data and created an interactive dashboard to uncover key business insights.

🛠️ Tools & Technologies
Power BI
Power Query
Data Cleaning & Transformation
Data Visualization
DAX (Data Analysis Expressions)

📌 Key Features
Interactive Dashboard
Total Transaction Analysis
Transaction Amount Tracking
State-wise Performance Analysis
Category-wise Transaction Insights
Dynamic Filters and Slicers
KPI Cards and Business Metrics

📈 Dashboard Insights
Identified top-performing states based on transaction volume.
Analyzed transaction amount trends across different periods.
Compared transaction categories to understand user behavior.
Visualized key metrics for better decision-making.
